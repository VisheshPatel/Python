### Iterating List
```
list = [3,4,5,6]

for index, value in enumerate(list):
        print index, value
```

### Iterating Tuple
```
tuple = [(1,2), (3,4), (5,6)]

for index, (a, b) in enumerate(tuple):
        print "Index = ", index, "First", a, "then", b
```
