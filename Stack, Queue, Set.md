
### List as Stack

```
stack = []

stack.append(3)	# Works As a Push Operation, Add at right side(that is bottom)
stack.append(2)
stack.append(1)

print(stack.pop())	# Pop Element From Last(that is bottom)
print(stack.pop())	
print(stack.pop())	

if stack == []:		# OR if not stack:
	print("Stack is Empty")
```
